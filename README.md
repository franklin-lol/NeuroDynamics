# NeuroDynamics Suite — Genesis Edition (v3.1)

Высокотехнологичный программный комплекс для генерации нейроакустических сессий и исследования глубоких состояний сознания. Проект переведен на модульную архитектуру для обеспечения максимальной точности синтеза и гибкости профилирования.

## Архитектура
Проект разделен на логические модули в пакете `psychoacoustic/`:
- **`dsp.py`**: Ядро синтеза (бинауральные пары, изохронные ритмы, 80 Hz ASSR, Schumann Stack).
- **`block.py`**: Объектная модель сессионного блока с поддержкой 12+ слоев воздействия.
- **`renderer.py`**: Многопоточный (поблочный) рендеринг в FLAC PCM-24 с Equal-Power кроссфейдами.
- **`profiles.py`**: Библиотека готовых архитектур сессий (GENESIS, SLEEP, ORACLE и др.).
- **`chaos.py` & `phi.py`**: Модули нелинейной динамики (аттракторы Лоренца) и геометрии Золотого Сечения.
- **`hrtf.py`**: Аппроксимация передаточной функции головы для внешней локализации звука.

## Ключевые технологии
*   **CFC (Cross-Frequency Coupling):** Фазово-амплитудная модуляция (тета-ритм управляет амплитудой гаммы).
*   **Stochastic Resonance:** Калиброванный шум (15% RMS) для снижения порога восприятия сигналов.
*   **Lorenz Chaos:** Детерминированный хаос в модуляции биений для предотвращения нейронной адаптации.
*   **ITD Rotation:** Медленная фазовая ротация для стимуляции вестибулярного аппарата.
*   **Infra-Modulation:** Сверхнизкочастотная амплитудная модуляция (0.025–0.1 Hz) для синхронизации с дыханием и ВНС.

## Использование

### Установка
```bash
pip install -r requirements.txt
```

### Генерация сессий
```bash
# По умолчанию генерирует флагманскую сессию GENESIS (63 мин)
python genesis_generator.py

# Генерация конкретного профиля
python genesis_generator.py --profile SLEEP

# Просмотр карты сессии (тайминги и слои) без рендеринга
python genesis_generator.py --profile ORACLE --map

# Генерация всех доступных профилей
python genesis_generator.py --all
```

## Доступные профили
- **GENESIS (63 мин):** Полный спектр: активация (Ignition) → Void → Интеграция.
- **SLEEP (90 min):** Циклическая дельта с "окнами осознания" (гамма-всплески).
- **WALK (30 min):** Альфа-тета расширение, оптимизированное для прослушивания в движении.
- **HEALER (75 min):** Глубокая регенерация, акцент на частоте 528 Гц и пропорциях Phi.
- **ORACLE (60 min):** Тета-навигация и резонанс Шумана.
- **WARRIOR (50 min):** Пиковая активация (бета/гамма) с использованием HRTF и хаоса.
