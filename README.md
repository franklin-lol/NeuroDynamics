# NeuroDynamics Suite

Программный комплекс для генерации нейроакустических сессий и исследования психоакустических эффектов. Инструментарий позволяет создавать высокоточные аудиофайлы в формате FLAC (PCM_24) с применением продвинутых техник воздействия на состояния сознания.

## Состав комплекса

### 1. Operator Matrix (operator_matrix_gen.py)
Специализированный генератор фиксированных сессий. Каждая сессия спроектирована под конкретную прикладную задачу:
- **Прогулка (30 мин):** поддержание состояния активного наблюдателя без погружения в транс.
- **Медитация (40 мин):** глубокое погружение с использованием логарифмических частотных свипов и несущей частоты 528 Гц.
- **Сон (120 мин):** архитектура "Окон Осознания" для работы в фазе глубокого сна и потенциальных OBE-практик.
- **Divine (45 мин):** максимальная конфигурация с использованием всех 5 гармоник частот Шумана.

### 2. Genesis Engine (genesis_generator.py)
Универсальный движок для генерации динамических сессий на основе профилей. Включает в себя каскады частот Сольфеджио и гибкую систему управления через командную строку.
Поддерживаемые профили:
- **GENESIS:** полный спектр от активации до глубокого погружения.
- **HEALER:** акцент на дельта-ритмах и восстановлении.
- **ORACLE:** работа с тета-состояниями и резонансом Шумана.
- **WARRIOR:** высокочастотная активация (бета/гамма).

## Технические особенности (v3.0)
- **Beat Jitter (1/f):** случайные микро-изменения частоты биений для предотвращения нейронного привыкания.
- **Equal-Power Crossfades:** использование √-затухания для сохранения постоянной мощности звука при переходах.
- **Cross-Frequency Coupling (CFC):** фазово-амплитудная модуляция (тета-ритм модулирует амплитуду гамма-волн).
- **ASSR 80 Hz:** независимый путь стимуляции ствола мозга и слуховой коры.
- **Spatial Rotation:** медленное вращение фазы для задействования вестибулярного аппарата.
- **Stochastic Resonance:** калиброванный шум для улучшения восприятия подпороговых сигналов.

## Использование

### Установка зависимостей
```bash
pip install -r requirements.txt
```

### Запуск генераторов
```bash
# Генерация набора Operator Matrix
python operator_matrix_gen.py

# Генерация сессии Genesis с определенным профилем
python genesis_generator.py --profile HEALER
```
