# NeuroDynamics Suite — Genesis Edition (v3.2)

Высокотехнологичный программный комплекс для генерации нейроакустических сессий и исследования глубоких состояний сознания. Версия 3.2 "Organic & Scientific" переводит проект из разряда математического синтеза в область высокохудожественного психоакустического ПО.

## Новое в версии 3.2
*   **Organic Sound Engine:** 4 тембральных режима (Warm, Rich, Soft, Organ) и новые слои:
    *   *Detuned Drone:* 3-голосный хорус-пэд с микро-LFO.
    *   *Resonant Wind:* Розовый шум, отфильтрованный по частоте несущей.
    *   *LFO Filter:* Динамическое "дыхание" сигнала через качающийся LP-фильтр.
*   **Scientific Spatial Model:** 
    *   *Brown-Duda HRTF:* Физически корректная модель сферической головы.
    *   *Woodworth ITD:* Точный расчет межушных задержек.
    *   *3D Orbital:* Орбитальное движение звука (азимут + элевация) по траекториям Лиссажу.
*   **Chaos Stabilization:** Исправлен временной масштаб аттрактора Лоренца (0.5–3 Hz) и введена полная детерминированность по Seed.
*   **Smart Rendering:** Автоматическая генерация **CUE sheets** для отображения названий фаз в плеерах (foobar2000, VLC, mpv).

## Архитектура
Проект разделен на логические модули в пакете `psychoacoustic/`:
- **`dsp.py`**: Ядро синтеза (тембры, дрон, ветер, LFO-фильтры, бинауральные пары).
- **`block.py`**: Объектная модель сессионного блока (40+ параметров воздействия).
- **`renderer.py`**: Рендеринг в FLAC PCM-24 с CUE-поддержкой и кроссфейдами.
- **`profiles.py`**: Библиотека архитектур сессий (GENESIS, SLEEP, HEALER и др.).
- **`hrtf.py`**: Аналитическая модель HRTF Брауна-Дуды.
- **`chaos.py`**: Модуль нелинейной динамики (аттракторы Лоренца).

## Ключевые технологии
*   **CFC (Cross-Frequency Coupling):** Тета-ритм управляет амплитудой гаммы ( hippocampal coupling).
*   **Stochastic Resonance:** Розовый шум, откалиброванный к 15% RMS сигнала.
*   **Lorenz Chaos:** Детерминированный хаос для деактивации DMN (сети пассивного режима мозга).
*   **Phase Lock:** Синхронизация частоты биений с ритмом дыхания (infra-modulation).
