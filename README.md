# NeuroDynamics Suite

Высокотехнологичный программный комплекс для генерации нейроакустических сессий и исследования глубоких состояний сознания. Проект представляет собой конвергенцию передовых психоакустических протоколов, математического моделирования и художественного саунд-дизайна, направленную на направленную модуляцию нейронной активности.

## Научный фундамент
NeuroDynamics Suite оперирует на стыке нейробиологии и акустики, используя механизмы **Neural Entrainment** для достижения специфических состояний:
*   **DMN Deactivation:** Использование хаотических аттракторов Лоренца для временной деактивации Default Mode Network, что способствует снижению "мыслительной жвачки" и входу в трансовые состояния.
*   **Hippocampal Coupling:** Реализация CFC (Cross-Frequency Coupling), где высокочастотные гамма-ритмы модулируются фазой тета-волн, имитируя естественные паттерны обработки информации в гиппокампе.
*   **Coherence Optimization:** Использование частот золотого сечения (φ=1.618) для создания гармонических структур с минимальным когнитивным сопротивлением.

## Технологическое ядро

### 1. Organic Sound Engine
Система синтеза отходит от стерильных осцилляторов в пользу "живого" тембрального наполнения:
*   **4 Synthesis Modes:** 
    *   *Warm:* Мягкий FM-синтез с богатыми четными гармониками.
    *   *Rich:* Аддитивный синтез (5+ слоев) для глубокого резонанса.
    *   *Soft:* Чистые синусоидальные формы для дельта-состояний.
    *   *Organ:* Сложные органоподобные структуры для Шумановских резонансов.
*   **Atmospheric Layers:** 
    *   *Detuned Drone:* Трехголосные пэды с микро-модуляцией, создающие эффект "стены звука".
    *   *Resonant Wind:* Динамический розовый шум, выполняющий роль несущей для стохастического резонанса.

### 2. Scientific Spatial Model (3D)
Полное погружение обеспечивается аналитическим расчетом звукового поля:
*   **Brown-Duda HRTF:** Физически корректная модель Head-Related Transfer Function, учитывающая дифракцию звука вокруг головы.
*   **Woodworth ITD:** Прецизионный расчет межушных временных задержек для формирования точного азимута.
*   **Lissajous Orbital:** Алгоритмическое движение источников по 3D-траекториям, предотвращающее привыкание мозга к стимулу.

### 3. Chaos & Nonlinearity
В отличие от линейных бинауральных ритмов, NeuroDynamics использует нелинейную динамику:
*   **Lorenz Chaos:** Детерминированный хаос в диапазоне 0.5–3 Hz, вносящий "органическую" нестабильность в ритмическую структуру.
*   **Jitter Modulation:** 1/f шум в огибающих биений для имитации биологической вариабельности.

## Архитектура сессий
Каждая сессия строится по принципу экспоненциального погружения:
1.  **Ignition (5–10 мин):** Захват внимания, синхронизация с внешним ритмом (Alpha/Beta).
2.  **Transition (15–20 мин):** Постепенное замедление несущих частот, активация CFC-протоколов.
3.  **Void Core (20–40 мин):** Максимальная глубина (Theta/Delta), активная работа хаос-аттракторов и 3D-пространства.
4.  **Integration (5–10 мин):** Возврат в рабочее состояние (Beta-lock), закрепление нейронных связей.

## Протокол использования
Для достижения максимального эффекта **необходимо** соблюдение следующих правил:
*   **Оборудование:** Только проводные наушники открытого или закрытого типа. Bluetooth-кодеки разрушают фазовую целостность ITD-сигнала.
*   **Громкость:** Оптимальный диапазон 15–30%. Звук должен быть "на грани восприятия", не вызывая дискомфорта.
*   **Положение:** Полулежа или лежа, глаза закрыты. Прямой позвоночник способствует более глубокому погружению.
*   **Регулярность:** Не чаще одного раза в 48 часов для предотвращения нейронной адаптации.
