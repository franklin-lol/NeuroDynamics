# NeuroDynamics Suite

[![Tests](https://img.shields.io/badge/Tests-Passed-brightgreen)](#)

Высокотехнологичный программный комплекс для генерации нейроакустических сессий и исследования глубоких состояний сознания. Проект представляет собой конвергенцию передовых психоакустических протоколов, математического моделирования и художественного саунд-дизайна, направленную на направленную модуляцию нейронной активности.

## Научный фундамент
NeuroDynamics Suite оперирует на стыке нейробиологии и акустики, используя механизмы **Neural Entrainment** для достижения специфических состояний:
*   **DMN Deactivation:** Использование хаотических аттракторов Лоренца для временной деактивации Default Mode Network.
*   **Hippocampal Coupling:** Реализация CFC (Cross-Frequency Coupling), где высокочастотные гамма-ритмы модулируются фазой тета-волн, имитируя естественные паттерны обработки информации в гиппокампе.
*   **Coherence Optimization:** Использование частот золотого сечения (φ=1.618) для создания гармонических структур с минимальным когнитивным сопротивлением.
*   **Respiratory Entrainment:** Синхронизация с ритмом дыхания (4–6 BPM) для стимуляции блуждающего нерва и входа в состояние когерентности HRV.

## Технологическое ядро

### 1. Multi-Pathway Sound Engine
Система воздействует на мозг через три независимых канала:
*   **Binaural Beats:** Фазовый сдвиг в HRTF-пространстве для воздействия на кору.
*   **Monaural Beats:** Физическая амплитудная модуляция (AM) для прямого влияния на стволовые структуры.
*   **Isochronic Priming:** Гамма-вспышки (FFR prime burst) для подготовки мозга к сессии.

### 2. Organic Synthesis & DSP
*   **4 Synthesis Modes:** Warm (FM), Rich (Additive), Soft (Pure), Organ (Drawbar).
*   **Formant Resonator:** Банк фильтров, имитирующих форманты человеческого голоса для органического тембра.
*   **Tube Saturation:** Мягкое аналоговое насыщение для добавления "тепла" и четных гармоник.
*   **Synthetic Room Reverb:** Декоррелированная реверберация для создания пространственной глубины.

### 3. Scientific Spatial Model (3D)
*   **Brown-Duda HRTF:** Физически корректная модель Head-Related Transfer Function.
*   **Woodworth ITD:** Прецизионный расчет межушных временных задержек.
*   **Lissajous Orbital:** Алгоритмическое движение источников по 3D-траекториям.

### 4. Chaos & Nonlinearity
*   **Lorenz Chaos:** Детерминированный хаос (0.5–3 Hz) в ритмической структуре.
*   **Jitter Modulation:** 1/f шум в огибающих для имитации биологической вариабельности.

## Архитектура сессий
Каждая сессия строится по принципу экспоненциального погружения:
1.  **FFR Prime (30 сек):** Начальная гамма-активация.
2.  **Ignition (5–10 мин):** Захват внимания, синхронизация с внешним ритмом (Alpha/Beta).
3.  **Transition (15–20 мин):** Плавное замедление несущих частот через профессиональный crossfade-тишину.
4.  **Void Core (20–40 мин):** Максимальная глубина (Theta/Delta), активная работа респираторных AM-слоев.
5.  **Integration (5–10 мин):** Возврат в рабочее состояние (Beta-lock).

## Протокол использования
*   **Оборудование:** Только проводные наушники. Bluetooth-кодеки разрушают фазовую целостность ITD-сигнала.
*   **Громкость:** Оптимальный диапазон 15–30%. Система использует заниженный порог нормализации (-4.0 dBFS) для комфорта.
*   **Положение:** Полулежа или лежа, глаза закрыты. Прямой позвоночник способствует более глубокому погружению.
*   **Регулярность:** Не чаще одного раза в 48 часов для предотвращения нейронной адаптации.

## Архитектура ПО
*   **`psychoacoustic/`**: Ядро обработки сигналов и математических моделей.
*   **`genesis_generator.py`**: Интерактивный терминальный интерфейс (Dashboard).
*   **`renderer.py`**: Высокоточный рендеринг в FLAC 24-bit / 44.1kHz с поддержкой метаданных.
*   **`tests/`**: Набор unit-тестов для проверки целостности DSP и уровней громкости.
